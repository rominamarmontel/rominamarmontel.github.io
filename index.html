About

Hiromi VARNIER

I have been studying Programming at Holberton School since January 2022 and chose an e-commerce site for my portfolio assignment. I had been asked to create a website by a friend who is an art appraiser. Considering the clientele, I tried to keep the site simple and easy to understand. My friend wants a site that not only sells art and antiques, but also provides estimates and appraisals, so we plan to add a page in the future that will allow users to easily estimate art and antiques via online video. This was my first attempt at JavaScript for the front and back of the site, but I was able to create it while looking at it comfortably to try out the many features, so I hope you all enjoy it as much as I did.

Post function

Easy to post in the admin panel
Easy to post and delete product information and images. You can also register the number of items in stock, so you don't have to worry about ordering out-of-stock items!

Project Name

The image page of each individual product has a stock availability indicator, a cart function, and the ability for only logged-in users to post product rating comments. The number of stars varies depending on the rating.

Security

Your personal information will be protected
User passwords are encrypted during registration and are securely protected. Also, after logging in, the user information can be changed after login.
